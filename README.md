## Installation

### Install compile dependencies
> libseccomp-dev
> build-essential
### Installtion for arch
```sh
yay -S chocolate
```


### Installtion chocolate with makefile

```sh
git clone https://github.com/frchocolate/chocolate
cd chocolate
make all
```

## Usage
Run the `chocolate version` command to get started.
